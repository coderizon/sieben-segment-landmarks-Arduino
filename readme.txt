Gestenerkennung nachtrainieren mit Model Maker
==============================================

Dieses Projekt zeigt, wie du ein Gestenerkennungs-Modell mit Google Model Maker lokal per Transfer Learning nachtrainierst. Eine CPU reicht aus, weil die komplette Modelllogik erhalten bleibt und nur die letzten Schichten neu trainiert werden. Das offizielle Google-Colab scheitert aktuell an Versionskonflikten, deshalb liefern wir eine lokal funktionierende Alternative.

Hinweis: Bitte nur mit Python 3.11 oder älter ausführen.
